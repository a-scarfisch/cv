Este es mi CV interactivo desarrollado como parte de mi formación en el **Bootcamp Full Stack Java Jr.**. Combina mi formación en **Literatura Creativa** con mis nuevas habilidades técnicas.

## 🚀 Características
- **Diseño Custom:** Estética Dark Mode con acentos en Morado Neón.
- **Responsivo:** Adaptado para móviles y escritorio usando **Bootstrap 5**.
- **Animaciones:** Implementación de la librería **AOS** para transiciones suaves.
- **Integración:** Enlaces directos a mi [Substack (p3ngu1n)](https://substack.com/@p3ngu1n21) y repositorios de GitHub.

## 🛠️ Tecnologías utilizadas
* **HTML5** (Estructura semántica)
* **CSS3** (Customización de estilos y efectos neón)
* **Bootstrap 5** (Layout y componentes como el Acordeón)
* **JavaScript** (Interactividad)
* **AOS Library** (Animaciones de scroll)

## 📖 Contexto Académico
Este proyecto representa la intersección entre las humanidades digitales y la arquitectura de software. Actualmente, curso el programa de Generation Chile mientras finalizo mi tesis sobre traducción localizada de Emily Brontë.
